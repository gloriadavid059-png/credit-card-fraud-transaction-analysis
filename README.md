# Credit Card Fraud & Transaction Analysis

## Project Overview

This project analyzes credit card transaction data using PostgreSQL and Excel to identify customer spending patterns, transaction trends, and fraud activity.

The dataset contains 100,000 transactions across multiple countries, card types, and merchant categories. SQL was used to extract business insights, and Excel was used to create an interactive dashboard for visualization.

---

## Tools Used

* PostgreSQL
* Microsoft Excel
* SQL
* Data Visualization

---

## Business Questions Answered

1. What is the total transaction value?
2. How many transactions occurred?
3. What is the average transaction amount?
4. What percentage of transactions were fraudulent?
5. Which countries generated the highest transaction value?
6. Which merchant categories experienced the highest fraud activity?
7. Which countries recorded the most fraud cases?
8. What are the most frequently used card types?
9. Who are the top 10 customers by spending?
10. How did transaction activity change over time?

---

## Key Findings

* Total Transaction Value: $5.96M
* Total Transactions: 100,000
* Average Transaction Amount: $59.59
* Fraud Rate: 2.57%
* USA generated the highest transaction value.
* France recorded the highest number of fraud cases.
* Clothing recorded the highest fraud activity.
* The highest-spending customer generated over $5,800 in transaction value.
